Class: MapItem

MapItem(name, mask, shift, valueMask)

Class represeting a MapItem

Constructor

new MapItem(name, mask, shift, valueMask)

More used as a structure to contain the masks and shifts used to get/set elements from a register.
Parameters:
Name Type Description
name String the name of the bit field, used when getting or setting the property
mask Number the mask to apply to the reg variable when getting the value and the bit wise not (~) mask when setting
shift Number the number of bits to shift right shift when getting left shift when setting
valueMask Number the mask to apply to the value when setting the reg variable making sure it stays the appropriate size
